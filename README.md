# 망가번역기

일본어 만화 이미지를 한국어로 가볍게 번역해서 **감상용으로 읽기 편하게 만드는 Windows 데스크톱 앱**입니다.

이 앱은 전문 식질/역질 툴이 아닙니다. 원본 페이지 위에 번역 텍스트 블록을 올려서 빠르게 읽는 용도에 맞춰 만들었고, 출판용 편집, 말풍선 지우기, 배경 복원, 폰트 연출, 레이어 합성 같은 식질 기능은 의도적으로 넣지 않았습니다.

이미지 몇 장, 폴더, ZIP 파일을 보관함에 추가해서 번역하고, 결과가 어색하면 화면에서 텍스트 블록과 문장만 간단히 고칠 수 있습니다. 작품 데이터를 다른 PC나 사용자에게 넘길 수 있는 공유/가져오기 기능도 제공합니다.

## 먼저 알아둘 점

번역 엔진은 2가지입니다.

- `OpenAI Codex`: OpenAI 계정으로 로그인해서 번역합니다. 고성능 그래픽카드가 없어도 사용할 수 있습니다.
- `Gemma 4`: 내 컴퓨터에서 직접 AI 모델을 실행합니다. 이 방식은 NVIDIA RTX GPU와 충분한 VRAM이 필요합니다.

처음 쓰는 분에게는 `OpenAI Codex`를 권장합니다.

- PC 사양을 덜 탑니다.
- 모델 파일을 직접 받을 필요가 없습니다.
- 기본 모델은 `gpt-5.5`입니다.
- 설정에서 생각 수준을 `없음`, `낮음`, `보통`, `높음`, `최고` 중에서 고를 수 있습니다.

`Gemma 4`는 인터넷 없이 로컬에서 돌리고 싶거나 OpenAI 계정을 쓰지 않고 내 PC에서 처리하고 싶을 때 선택하면 됩니다.

- NVIDIA RTX GPU 필요
- VRAM 최소 12GB 이상
- VRAM 16GB 이상 권장
- AMD/Intel GPU, 내장 그래픽, VRAM이 부족한 그래픽카드는 지원하지 않습니다.

## 처음 사용하는 방법

1. 앱을 실행합니다.
2. 오른쪽 위 `설정`을 엽니다.
3. `번역 엔진`에서 `OpenAI Codex` 또는 `Gemma 4`를 고릅니다.
4. `OpenAI Codex`를 쓰는 경우 Windows PowerShell에서 Codex CLI 로그인을 먼저 해둡니다.
5. 왼쪽 위 `번역`을 누르고 `이미지`, `폴더`, `압축파일` 중 하나를 선택합니다.
6. 새 작품을 만들지, 기존 작품에 추가할지 선택합니다.
7. 화 제목을 확인한 뒤 보관함에 추가합니다.
8. 오른쪽의 `이어서 번역` 또는 `전체 다시 번역`을 실행합니다.
9. 번역이 끝나면 페이지를 열어 필요한 문장과 텍스트 블록만 가볍게 수정합니다.

`작품 일괄 번역`은 별도 버튼으로 유지됩니다. 폴더 안의 ZIP 파일들과, 하위 폴더 안에 이미지가 들어 있는 폴더들을 여러 화로 읽어 순서대로 처리합니다.

## OpenAI Codex 연결 방법

`OpenAI Codex` 엔진은 앱 안에 API 키를 붙여 넣는 방식이 아닙니다. Windows에 로그인해 둔 Codex CLI 인증 정보를 사용합니다.

OpenAI 공식 Codex CLI 로그인 도움말: https://help.openai.com/en/articles/11381614-api-codex-cli-and-sign-in-with-chatgpt

### Windows에서 처음 연결하기

1. Node.js가 없다면 Node.js LTS 버전을 설치합니다.
   - 다운로드: https://nodejs.org/
   - 설치 후 PowerShell을 새로 열어야 `npm` 명령이 잡히는 경우가 있습니다.
2. Windows 시작 메뉴에서 `PowerShell`을 검색해 실행합니다.
3. Codex CLI를 설치합니다.

```powershell
npm i -g @openai/codex
```

4. Codex CLI 로그인을 실행합니다.

```powershell
codex --login
```

5. 브라우저가 열리면 OpenAI/ChatGPT 계정으로 로그인합니다.
6. PowerShell에서 로그인이 끝난 것을 확인한 뒤 망가번역기를 실행합니다.
7. 앱 오른쪽 위 `설정`에서 `번역 엔진`을 `OpenAI Codex`로 바꿉니다.
8. `Codex 모델`은 기본값 `gpt-5.5`를 그대로 두고, 필요하면 `생각` 수준을 조절합니다.
9. `잘 작동되나 확인`을 눌러 연결을 확인한 뒤 저장합니다.

전역 설치가 싫다면 아래처럼 한 번만 실행할 수도 있습니다.

```powershell
npx @openai/codex --login
```

그래도 앱에서 계속 쓰려면 Codex 로그인이 Windows 사용자 계정에 저장되어 있어야 합니다.

### 연결이 안 될 때

- `codex` 명령을 찾을 수 없다고 나오면 Node.js 설치 후 PowerShell을 새로 열고 `npm i -g @openai/codex`를 다시 실행하세요.
- 예전에 API 키 방식으로 Codex CLI를 쓴 적이 있다면 아래처럼 다시 로그인하세요.

```powershell
codex logout
codex --login
```

- 회사/학교 계정, Enterprise/Edu/Team 환경에서는 계정 정책에 따라 Codex CLI 로그인이나 사용이 제한될 수 있습니다.
- 앱의 `openai-oauth 포트`가 다른 프로그램과 겹치면 설정에서 포트를 다른 값으로 바꿔 저장하세요. 예: `10533`, `10534`, `10535`
- Windows 방화벽이나 백신이 로컬 연결을 물어보면 허용하세요. 앱은 `127.0.0.1` 로컬 주소로만 연결합니다.
- 번역 중 `Codex 로그인이 필요할 수 있습니다` 또는 모델 목록 확인 실패가 나오면 PowerShell에서 `codex --login`을 다시 실행한 뒤 앱에서 `잘 작동되나 확인`을 눌러보세요.
- 오류 원문은 앱 설정의 `로그 폴더 열기`에서 확인할 수 있습니다.

## 할 수 있는 일

- `번역` 버튼에서 이미지, 폴더, ZIP 압축파일을 보관함에 추가
- `작품 일괄 번역`으로 ZIP 여러 개나 여러 이미지 폴더를 한 번에 가져오기
- 새 작품으로 저장하거나 기존 작품에 이어 붙이기
- 작품/화 검색, 이름 변경, 순서 정리
- 페이지 순서 변경, 페이지별 재번역, 페이지 삭제
- `이어서 번역`, `전체 다시 번역`
- 번역 텍스트 블록 이동, 크기 조절, 문장 수정
- 작품 데이터 공유 파일 내보내기와 가져오기
- 공유받은 화를 새 작품으로 만들거나 기존 작품에 추가/교체/삭제 적용
- 설정에서 번역 엔진, 모델 프리셋, `GPU layers`, `NSFW 모드`, Codex 모델/생각 수준 변경
- 로그 폴더 바로 열기

## 의도적으로 하지 않는 일

이 앱은 만화 감상을 돕기 위한 번역 오버레이 앱입니다. 아래 기능은 목표 범위가 아닙니다.

- 원문 글자를 자연스럽게 지우는 인페인팅
- 말풍선/배경 복원
- 출판용 식자, 폰트 연출, 레이어 합성
- 완성본 이미지/PDF 제작
- 전문 역식 워크플로우 관리

번역 결과를 더 보기 좋게 맞추기 위한 텍스트 블록 수정 기능은 있지만, 목적은 “읽기 편하게 보기”에 가깝습니다.

## 공유와 가져오기

`공유하기`는 렌더된 이미지가 아니라 앱에서 다시 열 수 있는 작품 데이터 패키지를 만듭니다.

- 공유 파일 확장자는 `*.mgtshare`입니다.
- 선택한 작품과 화만 포함합니다.
- 원본 페이지 이미지, 번역 블록, 좌표, 스타일이 포함됩니다.
- 설정, 로그인 정보, 모델 파일, 로그, 분석 임시 파일은 포함되지 않습니다.

`가져오기`에서는 공유 파일을 새 작품으로 만들거나 기존 작품에 적용할 수 있습니다. 기존 작품에 적용할 때는 왼쪽의 최종 적용 목록과 오른쪽의 공유 파일 후보를 보면서 드래그로 추가하고 순서를 바꿀 수 있습니다. 기존 화를 삭제하거나 공유받은 화로 교체하는 흐름도 지원합니다.

## 설정 설명

앱은 기본으로 `Gemma 4` 기반 설정을 사용합니다. 설정 화면에서 `OpenAI Codex` 기반 번역 엔진으로 바꿀 수 있고, 변경 내용은 다음 번 번역부터 적용됩니다.

- 번역 엔진: `Gemma 4`, `OpenAI Codex`
- 모델 프리셋: `Q3`, `Q4`, `Q6`, `커스텀`
- 모델 소스: Hugging Face repo 또는 로컬 GGUF 파일
- Codex 모델 기본값: `gpt-5.5`
- Codex 생각 수준: `없음`, `낮음`, `보통`, `높음`, `최고`
- `GPU layers`: Gemma 4에서 GPU를 얼마나 사용할지 정하는 값
- `NSFW 모드`: 성인향 이미지 번역을 허용하는 설정

`OpenAI Codex` 엔진은 Codex CLI 로그인 정보를 사용합니다. 처음 사용할 때는 먼저 Windows PowerShell에서 `codex --login`으로 로그인해두세요.

## 저장 위치

- 개발 실행에서는 보관함이 `library/`에, 로그가 `logs/app.log`에 저장됩니다.
- 설치형 배포본에서는 실행 파일 옆 `data/` 폴더를 사용합니다.
- 보관함: `data/library`
- 로그: `data/logs`
- 설정: `data/settings.json`
- 모델 캐시: `data/hf-cache`

## 문제가 생겼을 때

번역이 안 되면 먼저 설정에서 어떤 엔진을 선택했는지 확인하세요.

- `OpenAI Codex`: PowerShell에서 `codex --login`을 다시 실행한 뒤, 앱 설정에서 `잘 작동되나 확인`을 눌러봅니다.
- `Gemma 4`: NVIDIA RTX GPU와 충분한 VRAM이 있는지 확인합니다.
- 앱이 멈춘 것처럼 보이는 경우: 큰 ZIP 파일이나 고해상도 이미지는 시간이 오래 걸릴 수 있습니다.
- 오류 내용을 확인하고 싶은 경우: 앱의 `로그 폴더 열기` 기능을 사용합니다.

프로그램을 직접 개발하거나 수정할 때만 필요합니다. 일반 사용자는 설치 파일을 실행하면 됩니다.

```powershell
npm install
npm run dev
```
